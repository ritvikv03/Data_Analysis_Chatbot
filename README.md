## Requirements

This app uses the following dependencies:

- `streamlit`
- `google-generativeai`
- `pdfplumber`
- `pandas`
- `python-docx`
- `openpyxl`
- `numpy`

These are all listed in the `requirements.txt` file.


```bash
 streamlit run chatbot.py
```
